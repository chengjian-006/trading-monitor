# 中继平台突破买入模型 BUY_PLATFORM_BREAKOUT — 设计 (2026-06-05)

## 目标与路线
- 把"中继平台突破"形态做成**独立新买点模型**(不动现有「缩量突破」)。
- 路线:**先回测后决定上线**。终点 = 像 N字 那样给一份"值不值得上 + 推荐参数 / 或不推荐理由"。
- 与现有「缩量突破 BUY_VOL_BREAKOUT」的边界:后者本质是"1日微型平台突破"(昨缩量+破昨高);本模型是**多日横盘平台**整理后突破上沿,口径不同。

## 活体样本 太极实业(600667)
真实日K(同花顺,列序 日期,开,高,低,收,量,额):
- 前置主升:4/30 收 9.59 → 5/20 收 13.69,14个交易日 +43%。
- 平台:5/21~6/1 共8个交易日,横在 12.1~14.6(收盘 12.42~13.92),平台上沿 PH≈14.60(5/27高)。
- 突破:6/2 收 14.94(量 1.29×平台均量)首破上沿 → 6/3 站稳 → 6/4 缩量冲 16.36。

两个由真实样本得到的硬约束(防止口径拍脑袋):
1. 平台振幅高达 ~20%(按高低)→ 平台振幅阈值若定 ≤8~12% 会漏掉太极。
2. 突破日放量仅 1.29× → 放量门槛若定到 2× 会漏掉太极。
→ 故所有阈值交给回测扫参,太极作为"必须能在 6/2 触发"的校验样本。

## 检测器口径(参数全部待回测扫)
- **中继前置**:平台起点前 N(≈20)日涨幅 ≥ R(扫 15/20/30%)。
- **平台**:最近 L 天(扫 6/8/12/15)横盘;平台上沿 PH=区间最高价max,下沿 PL=区间最低价min;
  振幅 (PH−PL)/PL ≤ A(扫 10/15/20%);期间收盘不破 MA20(不破位);可选平台期缩量。
- **突破触发(今日)**:收盘价(或盘中最高)> PH ×(1+buffer)(扫 0/0.5/1%);
  放量 ≥ 平台均量 × V(扫 1.0/1.3/1.5)。
- **当变量测、不预设**:突破口径(收盘确认 vs 盘中)、放量是否必须、中继前置是否必须。

## 回测 bt_platform.py
- 全市场近半年、bt_cache 日K、扣 0.30% 双边;与回踩MA10/MA20、缩量突破**同口径**。
- 扫参网格:L × A × buffer × V × 突破口径 × 持有窗(T+3/5/10)。
- 出场口径先用固定持有窗对比期望(后续可加 跌破MA5/MA10、止损 -8/-10%)。
- 输出:胜率 / 盈利因子PF / 净均值 / 最优持有 / 触发样本数;太极实业触发自检。

## 交付
"值不值得上"判断 + 推荐参数(或不推荐理由)。若 PF 站得住且不被现有模型覆盖 → 再议接线上线(signal_specs 三处登记、signal_id 前缀 BUY_)。

## 回测结论(2026-06-05, bt_platform.py, 全市场4957只)
**值得上 —— 明显强于现有买点。**

主配置(L8/收盘振幅≤15%/中继≥20%/收盘破PH+0.5%/放量1.2×, 出场同回踩MA10):
- 全部 n3676 **胜率69.0% 均值+4.30% PF3.08**;样本内70.5%/PF3.27;**样本外67.2%/PF2.85**(稳)。
- 对照:回踩MA10/MA20 ≈57%/PF1.9-2.0;缩量突破 65%/PF2.24。→ 完胜。

参数敏感性(全部稳健):
- 平台长 L6/8/12/15 → PF2.99/3.08/3.29/3.39(越长越少越精)。
- 收盘振幅 ≤10/15/20% → PF3.13/3.08/3.00。
- 放量 ≥1.0/1.2/1.5× / 不要求 → PF3.03/3.08/3.24/3.06(放量门槛影响很小)。
- 中继前置 ≥15/20/30% / 不要求 → PF2.90/3.08/3.01/**2.98**(核心edge不在中继, 它只是轻质量过滤: 加上把均值+3.76→+4.30%但样本砍半)。

**两条硬约束(落地关键):**
1. **必须收盘确认突破**:改"盘中破最高价"口径 → 胜率69%→**57.6%、PF3.08→1.72**(盘中假突破极多)。
2. **疑似吃上行市**:2025-06~2026-05 大体上行,12月仅2026-03(~50-55%/~0%)走平,其余+2~+7%。平台突破=顺势动量,**退潮/下行市大概率失效**→上线须与「大盘退潮风控」联动(退潮停发)。

太极实业自检:检测器在 600667 多个历史日触发(09-30/10-09/11-03/01-16/.../05-07/05-11), 确认能抓该形态;6/2突破因缓存末日5/29未入样。

**推荐上线参数**:收盘确认、L8~12、收盘振幅≤15%、中继前置≥20%(可选, 提质降量)、放量≥1.3×、成交额≥10亿;regime退潮时停发。
**出场口径(已定0605)**:沿用回踩MA10那套(+7%卖半/剩半收盘破MA10×0.98清/−6%止损/T+10封顶/10日去重), 与回测出场一致, 回测战绩直接适用。
