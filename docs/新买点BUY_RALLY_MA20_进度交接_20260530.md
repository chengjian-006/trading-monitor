## 二、调参全过程数据（怎么从硬币调到 1.9:1）

| 阶段 | 关键结论 |
|---|---|
| 原始(回踩MA20+主升浪+均额>20亿, 当日买) | 50/50 硬币: T+5 33%胜/36%亏 |
| +形态过滤(不破MA20/阳线/MA60) | 胜率纹丝不动, 无效 |
| +次日反包确认 | 35%→44%, 开始有效 |
| **+缩量回踩(<均0.8)** | **47%胜/1.65:1/+5.2%** — 缩量是 edge 真正来源 |
| 入场: 突破昨高即买 vs 收盘确认 | 突破即买盘中可操作, 但有假突破 |
| **突破幅度门槛** | >2.5%: 48触发/46%胜/1.91:1/+6.9%(选定); >2%:1.86; >3%:2.38但丢一个多氟多 |

**止盈优化结论（关键）**：任何止盈减仓都**降低**收益（信号靠肥尾大牛股赚钱, +10%卖半正好砍在起飞时）。T+5: 不止盈+7.0% > +20%卖半+6.4% > +10%卖半+5.4%。卖越早越多越伤。
