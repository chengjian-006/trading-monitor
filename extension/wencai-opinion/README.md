# 问财观点 · 浏览器扩展（股小察）

在同花顺问财用口语问一句，把「智能调度」的投顾式推荐存进股小察「问财观点」页。

## 给使用者（朋友）：怎么装

> 拿到扩展文件夹（或 zip，先解压）后，首次使用需配置一次上报令牌；扩展只连接 HTTPS 应用域名。

1. Chrome / Edge 地址栏输入 `chrome://extensions`（Edge 是 `edge://extensions`）回车
2. 打开右上角 **开发者模式**
3. 点 **加载已解压的扩展程序**，选中这个 `wencai-opinion` 文件夹
4. 装好后：
   - **前台用**：打开 iwencai（问财）任意页，右下角有个 💡 悬浮按钮，点它选问题即可，浮层会实时显示答案。
   - **后台用**：点浏览器右上角扩展图标 💡 → 弹窗里点预置问题或输入自定义问题，**不用开问财页**也能问（前提：这浏览器登录过问财、cookie 没过期）。

> 首次或长时间没用，如果提示「未登录」，去 iwencai 网站登录一次即可。

## 三种问法

- **问财页浮层**：iwencai 任意页右下角 💡 → 选问题。
- **弹窗**：点工具栏图标 💡 → 预置/自定义问题，后台跑。
- **右键提问**（v1.4.0 起）：任意网页选中一段文字 → 右键「用问财观点问：…」。在问财页走浮层，其他页后台跑。

## 设置（点扩展图标）

> 预置问题、定时、存档规则这些编辑量大的，建议点弹窗设置页顶部的 **「打开完整设置页」** 用宽屏编辑（v1.4.0 起）。两处共用同一份配置。

- **服务器地址**：内置为 `https://app.guxiaocha.com`；部署时须在扩展源码、Nginx 配置和 DNS 中一并替换为已签发 TLS 证书的应用域名，裸 IP 和 HTTP 不受支持。
- **我的昵称**：填了之后，观点卡会显示是谁问的。
- **预置问题**：每行一条，会出现在悬浮按钮菜单里。
- **深度研究**：默认关（普通模式每天 188 次）；开了更深但每天约 10 次额度。
- **问完自动存档 / 没识别出个股就不存**：按需勾。
- **定时自动问**：默认关。开了会在设定时间点后台自动问并存进股小察。⚠️ 这是自动化操作，理论上有被同花顺风控/封号的风险，自己权衡。

## 说明

- 首次上报时，扩展会要求输入管理员提供的上报令牌。管理员须将同一个高熵随机值配置到已忽略的 `config.json` 的 `wencai_opinion.ingest_token`；令牌保存在浏览器配置中，可被有本机浏览器资料访问权的人提取，因此仅用于阻止匿名远程滥用，不是不可导出的密钥。
- 这是 **LLM 投顾观点**，不是回测背书的买卖信号，仅供参考。
- 用的是你自己的问财登录，数据只发往你自己的股小察服务器。

## 自动更新（v1.3.0 起）

开发者模式装的扩展 Chrome 不会自动更新，所以内置了「版本检查」：

- 后台每 6 小时（及浏览器启动时）向服务器查一次最新版，发现新版就在**扩展图标打红点角标**并弹一次系统通知。
- 打开弹窗，顶部会出现 **「发现新版 → 下载更新」** 横幅，点它直接下载最新 zip；设置页底部也有 **「检查更新」** 按钮和当前版本号。
- 下载后按上面的步骤重新「加载已解压」覆盖即可（Chrome 会保留你的设置）。

> 服务器分发接口：`/api/wencai/ext/version`（查最新版）、`/api/wencai/ext/download`（下载 zip）。版本真源直接读服务器上已部署的 `manifest.json`，部署即同步、不会漂。

## 只想在问财页问一句？用油猴脚本（自动更新）

旧版篡改猴脚本使用的裸 IP/HTTP 分发地址已停用；请使用本扩展的 HTTPS 分发包。功能是问财页浮层问答 + 存档，没有扩展的后台跑/定时/弹窗。

## 给管理员：怎么打分发包

观点上报接口需要独立 token 鉴权。发布前生成高熵随机令牌，配置到服务器忽略的 `config.json`，并通过受控渠道将其提供给扩展使用者；不要把令牌写入 `manifest.json`、源码或发布包。发布新版流程：改代码 → 升 `manifest.json` 的 `version` → 部署（`extension/` 随包上服务器）→ 旧版用户会自动收到红点提醒。
