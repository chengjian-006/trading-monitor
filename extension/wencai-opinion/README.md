# 问财观点 · 浏览器扩展（股小察）

在同花顺问财用口语问一句，把「智能调度」的投顾式推荐存进股小察「问财观点」页。

## 给使用者（朋友）：怎么装

> 拿到扩展文件夹（或 zip，先解压）后装完即用，**零配置**，不用改任何东西。

1. Chrome / Edge 地址栏输入 `chrome://extensions`（Edge 是 `edge://extensions`）回车
2. 打开右上角 **开发者模式**
3. 点 **加载已解压的扩展程序**，选中这个 `wencai-opinion` 文件夹
4. 装好后：
   - **前台用**：打开 iwencai（问财）任意页，右下角有个 💡 悬浮按钮，点它选问题即可，浮层会实时显示答案。
   - **后台用**：点浏览器右上角扩展图标 💡 → 弹窗里点预置问题或输入自定义问题，**不用开问财页**也能问（前提：这浏览器登录过问财、cookie 没过期）。

> 首次或长时间没用，如果提示「未登录」，去 iwencai 网站登录一次即可。

## 设置（点扩展图标）

- **服务器地址**：内置写死，界面不展示、无需配置。
- **我的昵称**：填了之后，观点卡会显示是谁问的。
- **预置问题**：每行一条，会出现在悬浮按钮菜单里。
- **深度研究**：默认关（普通模式每天 188 次）；开了更深但每天约 10 次额度。
- **问完自动上报 / 仅识别出个股才上报**：按需勾。
- **定时自动问**：默认关。开了会在设定时间点后台自动问并上报。⚠️ 这是自动化操作，理论上有被同花顺风控/封号的风险，自己权衡。

## 说明

- 这是 **LLM 投顾观点**，不是回测背书的买卖信号，仅供参考。
- 用的是你自己的问财登录，数据只发往你自己的股小察服务器。

## 给管理员：怎么打分发包

观点上报接口已不做 token 鉴权（2026-07-16 起），直接把整个 `extension/wencai-opinion/` 文件夹打包成 zip 发出去即可，无需任何预填。
